# PairShot Icon Assets

This folder contains the finalized icon source and export assets generated from:

- `/Users/kkk/Documents/pair-shot-icon.svg`

## Source

- `source/pair-shot-icon.svg`: original full icon (rounded white tile + shutter symbol)

## Android Adaptive

- `android/adaptive/pair-shot-adaptive-background.svg`: solid off-white background (`#F2F3F7`)
- `android/adaptive/pair-shot-adaptive-foreground.svg`: symbol-only foreground (inset for adaptive safe area)
- `android/adaptive/pair-shot-adaptive-monochrome.svg`: symbol-only monochrome foreground

## Android Exports

- `android/play-store/pairshot-playstore-512.png`: Play Store listing icon
- `android/exports/legacy/`: launcher legacy size exports
- `android/exports/adaptive-foreground/`: adaptive foreground layer exports
- `android/exports/adaptive-monochrome/`: adaptive monochrome layer exports
