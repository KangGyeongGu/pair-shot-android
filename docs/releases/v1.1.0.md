# v1.1.0

`출시 예정`

## 신규 기능

### 쿠폰 시스템
- Short code 등록으로 광고 제거 기능 활성화 (서버 검증)
- 내 쿠폰 리스트 조회 (기기별)
- 쿠폰 등록 기간 자동 합산
- Revoke·Expired 상태 서버 배치(pg_cron)로 자동 동기화
- 네트워크 장애 대비 pending 재시도 큐

### 광고 배치
- AdMob 기본 통합: 상단 Adaptive Banner, 내보내기 완료 시점 Interstitial
- **Rewarded Video 게이트**: 워터마크·합성 상세 설정 진입 시 광고 시청 후 세션 잠금 해제
- **Native Ad 리스트 삽입**: Home·앨범 상세의 페어 리스트에 6개마다 2×1 광고 카드 삽입
- **App Open 광고**: 앱 포그라운드 진입 시 자동 노출 (콜드 스타트 3초 대기 포함)
- **전면 광고 상호 배제**: `FullscreenAdState` 코디네이터로 Interstitial·Rewarded·AppOpen 3종 중 동시 1개만 노출
- 쿠폰 활성화 사용자는 전 광고 자동 스킵

### 카메라 회전 가이드
- Before 사진이 가로로 촬영된 경우 After 촬영 시 기기가 세로이면 중앙에 회전 힌트 노출
- `Icons.Filled.RotateLeft` / `RotateRight` + "왼쪽/오른쪽으로 눕혀 주세요"
- 페이드인 → 유지 → 페이드아웃과 0~90° 회전을 동반한 부드러운 반복 애니메이션
- 오버레이 토글 상태와 독립 작동 (`readBeforeRotation` 경량 경로 분리)

## 버그 수정

- **완성된 페어 재촬영**: 합성 미리보기 모달에서 재촬영 진입 시 대상 페어가 로드되지 않고 다른 페어가 잘못 표시되던 문제 수정
- 재촬영 시 대상 페어만 스트립에 단독 표시, 저장 완료 후 즉시 뒤로가기
- 오버레이 설정 저장, After 카메라 노출 경로, 개인정보 URL 보조 수정

## 내부 개선

- 빌드 `versionName`·`versionCode`를 git 태그 기반으로 자동 주입
  - `git describe --tags` → `{major}.{minor}.{patch}` 및 `major * 10_000 + minor * 100 + patch`
- 쿠폰 시스템 보안·아키텍처 전면 리팩토링 (서버 단일 검증 전환)
- `AfterCameraViewModel`의 retake resolution race 조건 해결 (`_retakeResolved` 플래그)
- 오버레이 rotation 계산을 EXIF-only 경량 경로로 분리해 슬라이더 드래그 시 중복 I/O 감소
- `OverlayTransformCalculator`에 `LANDSCAPE_LEFT_ROTATION` / `LANDSCAPE_RIGHT_ROTATION` 공개 상수 노출
- 정밀 검수·Detekt·Lint 이슈 전수 해소
