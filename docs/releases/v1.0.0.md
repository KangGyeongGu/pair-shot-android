# v1.0.0

`2026-04-24`

PairShot 최초 Play Store 출시 버전입니다.

## 주요 기능

### 촬영·페어링
- CameraX 기반 Before·After 전용 카메라 흐름
- Before 촬영 후 자동 페어링, After 촬영 시 동일 구도 오버레이 가이드
- 핀치 줌·플래시·야간 모드·HDR·격자·수평계 지원

### 앨범 관리
- 페어 카드 그리드 뷰 (Home·앨범 상세)
- 앨범 생성 (위치 정보 자동 기록), 이름 변경, 삭제
- 다중 선택 모드로 일괄 공유·저장·삭제·내보내기

### 합성·워터마크
- 선택한 페어를 단일 비교 이미지로 합성 (좌우·상하 레이아웃, 테두리·레이블 커스터마이즈)
- 텍스트·로고 워터마크 위치·크기 커스터마이즈 후 자동 삽입

### 내보내기·공유
- MediaStore 기반 갤러리 저장
- ZIP 압축 내보내기 (여러 페어 일괄)
- 시스템 공유 시트 연동

### 기타
- 한국어·영어 언어 지원
- 시스템 테마 연동 (라이트·다크·시스템)
