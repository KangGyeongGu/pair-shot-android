# v1.1.5

`2026-04-28`

## 사용자 체감

- **원본 사진이 갤러리에서 사라진 페어를 안전하게 처리**: 갤러리 앱에서 Before 또는 After 원본을 삭제해도 앱이 더 이상 깨지지 않음. 페어카드는 좌(Before)·우(After) 자리를 고정하고, 사라진 쪽만 빈 슬롯 placeholder 로 표시. 양쪽이 모두 사라지면 페어 자체가 자동 정리됨.
- **사라진 원본을 그 자리에서 재촬영**: 빈 슬롯이 있는 페어카드를 누르면 미리보기 화면에서 빈 쪽만 다시 촬영하는 모드로 진입. 1 장 촬영 후 카메라가 자동 종료되어 즉시 결과 확인 가능.
- **After 만 남은(AFTER_ONLY) 페어카드 탭 시 미리보기를 건너뛰고 곧장 Before 재촬영 카메라로 진입**: 한 단계 줄어든 동선.
- **홈·앨범 그리드에서 아래로 당겨 새로고침(Pull-to-Refresh)** 으로 갤러리 변경사항을 수동 동기화 가능. 앱을 백그라운드에서 다시 포그라운드로 가져올 때도 자동 동기화 1 회 실행.
- **합성 미리보기 안정화**: 16MP 이상 원본을 미리보기로 합성할 때 OOM 으로 까맣게 보이거나 앱이 멈추던 문제 해결. 합성 실패 시 에러 메시지와 "다시 시도" 버튼 표시.
- **카메라 화면 레이아웃 반응형 보정**: AdFree 등으로 광고 배너가 사라졌을 때 프리뷰 영역이 그 빈 공간을 자동으로 흡수해 더 크게 표시.
- **저장·공유 직전 자동 정리**: 일괄 저장/공유 시 사라진 원본을 미리 감지·제거한 뒤 진행하므로 중간에 실패하지 않음.
