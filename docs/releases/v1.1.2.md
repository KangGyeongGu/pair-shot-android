## 버그 수정

- **쿠폰 사용자에게 리워드 광고 다이얼로그가 뜨던 문제 수정**: 쿠폰으로 광고가 제거된 상태에서 워터마크·합성 상세 설정 버튼을 눌렀을 때, "광고 시청 후 사용 가능" 모달이 잠깐 노출되고 "광고 보기"를 눌러야 설정이 열리는 혼란스러운 UX를 제거.
- 이제 쿠폰 활성 사용자는 **모달 없이 즉시 상세 설정 진입**. 모달은 쿠폰 미활성·세션 내 미해제 사용자에게만 표시.
